Шапка сайта

  • Гиперссылка оформляется таксим образом:

    • <a href="https://gb.ru/lessons/265835">Конспект. Урок 2. HTML, CSS</a>

  • Шрифт оформляется таксим образом:

    • <link href="https://fonts.googleapis.com/css2?family=Roboto:wght@100&display=swap" rel="stylesheet">

  • Связь с файлом CSS оформляется таксим образом:

    • <link href="NEWfile.CSS" rel="stylesheet" typr="text/css">

Фото

Фото оформляется таксим образом:

<img height="150" src="https://www.interviewbit.com/blog/wp-content/uploads/2021/10/HTML-and-CSS.png" 

        alt="Мое тестовое изображение">

Заголовки

Заголовки оформляются таким образом:

<h1> Главный заголовок </h1>
 
<h2> второй заголовок </h2>
 
<h3> третий заголовок </h3>
 
<h4> четвертый заголовок </h4>

Главный заголовок

Второй заголовок

Третий заголовок

Четвертый заголовок

И так далее.

Списки

Списки оформляются таким образом:

<ul>
            <li>Лиса</li>
            <li>Каспер</li>
            </ul>
  • Лиса
  • Каспер
<ol>
        <li>Один</li>
        <li>Два</li>
        <li>Три</li>
         </ol>
  1. Один
  2. Два
  3. Три

Стили

Лучше стили оформлять в css файле, но в html оформление такое (где li и h4 - это нужные теги):

<style>
    li, h4 {
        color: black;
        font-family: 'Roboto', sans-serif;
    }

    </style>
